# vanilla-project

## Project setup
```

npm install
```

### Compiles and hot-reloads for development
```

npm start
```

### Compiles and minifies for production
```

npm run build
```

### Feature
```

- Vanilla JS
- ES6 with Babel
- CSS
- Module bundler width Webpack
```

### Description
```

Vanilla JS로 검색 폼 구현하기

기능
- 검색 폼
- 리셋 버튼
- 추천검색어
- 최근 검색어
- Intersection Observer를 이용한 무한 스크롤 구현 

데이터
- Kakao Open API 이용

```
